# AI-Detect-FruitVegetable
I tried to create AI that can detect 5 Fruits and 5 Vegetables
Fruit that can be detected:
-> Anggur   (Grapes)
-> Apel     (Apple)
-> Jeruk    (Oranges)
-> Manggis  (Mangosteen)
-> Pisang   (Banana)

Vegetable that can be detected:
-> Brokoli  (Broccoli)
-> Kentang  (Potato)
-> Kol      (Cabbage)
-> Terong   (Eggplant)
-> Wortel   (Carrot)
